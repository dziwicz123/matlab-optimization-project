# MATLAB Optimization Project

This project presents numerical optimization methods implemented in MATLAB.

## Problem
The goal is to find the point on the curve:

f(x) = 7 - x^(5/2)

that minimizes the distance to the point:

P = (4.5, 8)

## Methods used
- symbolic optimization using vpasolve
- numerical optimization using fminbnd
- custom implementation of the bisection method
- Newton–Raphson method for cylinder surface minimization

## Results
All numerical methods converge to:

x ≈ 0.885

The optimal point on the function is approximately:

(x, y) ≈ (0.885 , 6.263)
